== Description ==

fTourism is a Child Theme of fKidd Theme. The theme is Fully Responsive,
Built-in Slider, Multi-Level Drop-Down Custom Menu, Admin Panel Options,
W3C Markup Validated, Search Engine Optimized, Multiple Browsers support (IE9,
IE10, IE11, Chrome, Firefox, Safari, and Opera), FontAwesome Icons,
5 Widgets Areas: Sidebar, Homepage Column #1, Homepage Column #2,
Footer Column #1, and Footer Column #2, Custom Post Formats,
Page navigation, Threaded Comments, Translation Ready, and much more.


== Installation ==

Manual installation:

1. Upload the fTourism and fKidd folders to the /wp-content/themes/ directory

Installation using 'Add New Theme'

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for 'fTourism'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Theme Options to change theme specific options
